# Birthday Website Project

## Features
- **User Registration/Login:** Allows users to create an account and log in.
- **Birthday Reminders:** Users can set reminders for upcoming birthdays.
- **Event Planning:** Tools to help users plan birthday parties and events.
- **Customization Options:** Users can customize their profile and event settings.
- **Responsive Design:** Works on both desktop and mobile devices.

## Setup Instructions
1. **Clone the Repository:**  
   Clone this repository using the following command:
   ```bash
   git clone https://github.com/harshu-pixel/bday-web.git
   ```

2. **Navigate to the Project Directory:**  
   Change to the project directory:
   ```bash
   cd bday-web
   ```

3. **Install Dependencies:**  
   Install the required dependencies using npm:
   ```bash
   npm install
   ```

4. **Start the Application:**  
   Start the application using:
   ```bash
   npm start
   ```

5. **Open in Browser:**  
   Open your web browser and go to `http://localhost:3000` to view the site.

## Customization Guide
- **Change Styles:** Modify the CSS files in the `styles` directory to change the appearance of the website.
- **Add Features:** Feel free to modify the JavaScript files to add new features or change existing ones.
- **Update Configuration:** Change settings in the `config.js` file to customize the app's behavior.
